# 🚀 Quick-AI ✨

**Live Demo 👉** [https://quick-ai-ritu.vercel.app/ai](https://quick-ai-ritu.vercel.app/ai)

A powerful AI-powered web application that allows users to:

- 📝 Generate Articles using AI
- 🖼️ Generate AI Images (3D Style)
- 🧠 Get Smart Blog Titles
- 🧽 Remove Image Background
- 🧼 Remove Objects from Images
- 🗣️ Interact in the Community with Like functionality
- 📄 Upload Resumes for Instant Review
-  📊 Access a personalized **Dashboard** to view and manage all your previous tasks 

---

## 📸 Features

### 🔹 AI Tools
- 📝 **Article Generator** – Just type a prompt and get a full article instantly, with smart length filters: **Short (500–800 words)**, **Medium (800–1200 words)**, or **Long (1200+ words)** — crafted by AI to match your content goals.

- 🧠 **Blog Title Generator** – Instantly generate catchy, SEO-friendly titles based on your blog ideas and selected category: **General**, **Technology**, **Business**, **Health**, **Lifestyle**, **Education**, **Travel**, or **Food**.

- 🖼️ **Image Generator** – Generate stunning images in styles like **Realistic**, **Ghibli**, **Anime**, **Cartoon**, **Fantasy**, **3D**, and **Portrait** — and easily make your generated images public to showcase or share with others.

- **Background Remover** – Remove  backgrounds from uploaded images.
- **Object Remover** – Remove unwanted objects from images.
- **Resume Reviewer** – Upload your resume and get instant AI feedback.

### 🔹 Community
- View other users' generated content
- ❤️ Like your favorite creations.
- 🚀 Fully responsive and interactive UI.

---

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** Sql
- **Authentication:** Clerk
- **Cloud Services:** Cloudinary (for image storage), Vercel (for deployment), Neon for data storage.
- **AI Integration:** OpenAI API, Clipdrop API
- **Image Upload:** Multer middleware
---
## 🔐 Authentication

- Only logged-in users can access the features.
- Users can access features based on their subscription and free plan.
